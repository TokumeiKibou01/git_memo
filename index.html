Gitのメモ

基本

git init … 新しいリポジトリを作成するコマンド。

git add . … 今いるフォルダのすべてのファイルをgitに追加する（ステージする）コマンド。

git add "ファイル名" … 指定のファイル名をgitに追加する（ステージする）コマンド。

git commit -m "メッセージ" … 追加した（ステージした）ファイルを、確定させる（コミットする）コマンド。

git push … リモート（github.com等）に上げるコマンド。

git push -f … リモート（github.com等）に強制的に上げるコマンド。（コミットの内容によっては、-fを付けないとできない場合がある。）

git pull … リモート（github.com等）のやつをローカルに反映させるコマンド。

よく使うコマンド（※練習中）

git checkout ブランチ名 … ブランチを切り替えるコマンド。

git rebase -i コミットID … コミットを修正するときに使うコマンド。（コミットを統合させる、コミットメッセージを編集などがある。）

git stash … 今変更している内容を、一時的に避難させるコマンド。（どこかでも変更してると、一部コマンドが使えない。）

git stash pop … stashしたコマンドを、ローカルに反映させるコマンド。
